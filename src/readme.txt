------------------
 Golden Armor fix
------------------

Fixes the hidden golden plate armor's visibility when worn by an entity or the player.

--------------
 Installation
--------------

Unpack the contents of this zip and add it into
your Arx Fatalis/Arx Libertatis game's root folder

-----------
 Uninstall
-----------

Remove the following files from your Arx Fatalis/Arx Libertatis game's root folder:
- game/graph/obj3d/interactive/npc/human_base/tweaks/human_plate_cm.ftl

Or if you don't have any other mods installed then just delete the "graph" folders
